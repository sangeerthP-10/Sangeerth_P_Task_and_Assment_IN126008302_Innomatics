# Resume Analyzer using LLM (LangChain + HuggingFace)

## 🚀 Project Overview
This project analyzes resumes against job descriptions using LLM pipelines.

## 🔧 Features
- Resume Information Extraction
- Skill Matching
- Resume Scoring
- Explanation Generation

## 🛠 Tech Stack
- Python
- LangChain
- HuggingFace Transformers
